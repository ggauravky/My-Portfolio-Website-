## 🚀 About This Project

Welcome to my personal portfolio and blog website! This is a modern, responsive web application built to showcase my skills, projects, and technical writings in **web development**, **data science**, and **AI/ML**. 

### ✨ Key Highlights
- 🎨 **Clean & Modern Design** - Intuitive user interface with smooth animations
- 📱 **Fully Responsive** - Optimized for all devices (mobile, tablet, desktop)
- 🌙 **Theme Switching** - Dark/Light mode with user preference persistence
- 🔍 **Dynamic Content** - JSON-driven content management for easy updates
- ⚡ **Fast Performance** - Lightweight vanilla JavaScript implementation

## 🚀 Quick Start Guide

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local HTTP server (required for JSON file loading)

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ggauravky/My-Portfolio-Website-.git
   cd My-Portfolio-Website-
   ```

2. **Start a local server:**
   
   **Option A - Python:**
   ```bash
   python -m http.server 8000
   ```
   
   **Option B - Node.js:**
   ```bash
   npx http-server
   ```
   
   **Option C - VS Code:**
   - Install "Live Server" extension
   - Right-click on `index.html` → "Open with Live Server"

3. **Access the website:**
   ```
   http://localhost:8000
   ```
